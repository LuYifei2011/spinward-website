# 启旋游戏工作室成员信息登记与管理办法

（第一版·2026年5月发布）

## 第一条 目的

依据《启旋游戏工作室公约》第八条、第九条，制定本办法，规范成员信息登记流程，保障信息安全。

## 第二条 登记方式

本工作室成员登记分为**线上登记**和**线下登记**两种方式。

- **线上登记**：使用 Microsoft Form 表单（链接由管理部提供），适用于日常加入。
- **线下登记**：使用纸质表格，适用于面对面活动、管理职务任命。

## 第三条 线上登记字段及要求

线上登记表单包含以下字段。带“\*”的为必填项，其余为选填项。

| 字段                 | 必填 | 填写要求                                                        |
| -------------------- | ---- | --------------------------------------------------------------- |
| 工作室主用网名       | 是   | 不得含有违法、侮辱、色情、政治敏感或无意义的非常规字符。        |
| 其他常用网名         | 否   | 可列出其他平台使用的网名，方便成员识别。                        |
| 真实姓名             | 否   | 如担任管理职务或参加线下活动，须补充。                          |
| 出生日期             | 是   | 公历日期，格式 yyyy/m/d。本工作室保留要求提供年龄证明的权利。   |
| 联系方式             | 是   | 须注明联系方式类型及账号，如“QQ：123456789”或“微信：wxid_xxx”。 |
| 紧急联系人及联系方式 | 否   | 示例：“父亲 XXX 手机138xxxx”或“母亲 XXX 微信 Lisi123”。         |
| 本人承诺             | 是   | 复选框：已阅读并同意《启旋游戏工作室公约》及本办法。            |

**注**：登记日期由表单系统自动记录时间戳，无需填写。

## 第四条 线下登记字段及要求

线下登记使用纸质表格，除线上字段外，增加现场核验及监护人签名。

| 字段                 | 必填     | 填写要求                                                                   |
| -------------------- | -------- | -------------------------------------------------------------------------- |
| 工作室主用网名       | 是       | 不得含有违法、侮辱、色情、政治敏感或无意义的非常规字符。                   |
| 真实姓名             | 是       | 须与身份证件一致。                                                         |
| 出生日期             | 是       | 须与现场核验的身份证件一致。                                               |
| 联系方式             | 是       | 同线上。                                                                   |
| 紧急联系人及联系方式 | 条件必填 | 若成员未满十八周岁且参与线下活动，须提供监护人联系方式（活动结束后销毁）。 |
| 本人签名             | 是       | 手写签名。                                                                 |
| 监护人签名           | 条件必填 | 若成员未满十八周岁且参与线下活动，须监护人签署。                           |
| 登记日期             | 是       | 填写年/月/日。                                                             |

## 第五条 网名管理

成员可随时向管理部申请变更“工作室主用网名”，需说明变更理由。同一成员**每月变更网名不得超过三次**；超过三次的，管理部可要求其选择一个固定昵称并锁定三个月。管理部有权要求修改含有违法、侮辱、色情、政治敏感或无意义的非常规字符的网名。

## 第六条 信息保管与删除

**保管**：线上表单数据由室长及副室长加密保存（如设置密码的压缩包或加密云存储）；纸质表格锁入专用文件柜。管理部成员不得私自复制、转发成员信息。

**删除**：成员退出工作室后，其个人信息应在**三十日内**删除。项目贡献记录可匿名化保留（仅保留网名及贡献内容，删除真实姓名、联系方式等）。监护人可书面要求查阅或删除其子女信息，管理部应在**七个工作日内**处理。

## 第七条 违规处理

任何成员未经授权泄露其他成员信息，视为严重违反公约，依据《启旋游戏工作室成员行为守则》处理。恶意冒用他人信息提交登记者，工作室保留追究法律责任的权利。

## 第八条 生效

本办法自发布之日起生效。
